# 027_VIDEO_SCRIPT_STEERING.md

## Purpose
Govern educational videos, demonstrations, webinars, interviews, and portfolio walkthroughs.

## Video Types
Two-minute introduction, five-minute case study, architecture walkthrough, technical tutorial, service explainer, and recorded presentation.

## Script Structure
Audience problem, promise, context, visual roadmap, explanation, demonstration or evidence, limitations, takeaway, and CTA.

## Visual Rules
Use readable diagrams, captions, code zooms, and clear transitions. Remove credentials, private URLs, personal data, and confidential architecture details.

## Delivery Tone
Calm, direct, technically confident, and accessible. Define acronyms. Avoid excessive claims and cinematic hype.

## Validation
Technical demonstrations must be reproducible or clearly labeled as conceptual. Confirm audio, captions, links, and disclosure status before publication.
